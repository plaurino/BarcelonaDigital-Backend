## Prerequisites

You will need the following things properly installed on your computer.

* [Git](http://git-scm.com/)
* [Node.js](http://nodejs.org/) (with NPM), [Bower](http://bower.io/) and [Angular Cli](http://ngcli.github.io/)

## Installation

* `git clone <repository-url>` this repository
* change into the new directory
* `npm install`
* `bower install`

## Running / Development

* `npm start` or `npm development`
* Visit your app at [http://localhost:3080](http://localhost:3080).

### Code Generators

Generator helps you in generating controllers, directives,initializers, filters, services and factories right from your command line.
You can run:

* `ng generate directive <name>`
* `ng generate initializer <name>`
* `ng generate filter <name>`
* `ng generate service <name>`
* `ng generate factory <name>`

### Running Tests

* `ng test`

### Building

* `ng build`
